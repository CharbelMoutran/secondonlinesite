Why Learn HTML?

HTML, programming language, is essential for designing websites. It is a must-know for all programmers. This is why I decided to learn it!

By the way, did you know that HTML was originally designed by scientist for exchanging information about multiple topics? This is why we can write things like H2O and (x+y)3 which are scientific "formulas" I might say.Take a look at this:

The origins

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML.

HTML evolved over the years, and new standars were implemented, now HTML5 is based on well... HTML, the DOM (document object model), CSS and Javascript

Other useful langauges

Though HTML is essential for web development, you also need to master HTML and Javascript.

All in all, HTML is a very powerful language that also happens to be essential for web development so you should learn it
